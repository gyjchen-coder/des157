Structure,Prasing and Display

Structural elements are block elements that define block level elements like p, header, h1, h2, h3, footer tags. Inline elements only applies to specific words or phrases.

The different display settings include flexbox and grid.

Box Model

Box model treats each html element as a rectangular box that has 4 layers, content, padding, margin, and border. Paddings are space between the the text and box-border. Margin is space outside of border. Sometimes padding and margin can add additional peripheral space and create issues with gutters. To prevent this box-sizing property can be used to keep added paddings within pre-determined box size

Background Images

If the semantic tag of the image is important it's better to place it in html file for accessibility reason. If the image is purely decorative, such as background images, then placing it in CSS might allow better control of the image.
